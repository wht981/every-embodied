# 达摩院 & Datawhale 线上组队学习

## 主题

`0基础入门具身智能：4天极速闯关`

## 适合谁学

- 第一次系统接触具身智能的学习者
- 对机器人、大模型、自动驾驶、智能体感兴趣的同学
- 希望先建立整体认知，再决定是否继续深入代码和真机的开发者

**核心价值：**

> ✅ **零门槛**：小白友好的学习设计，轻松入门具身智能

> ✅ **真验证**：融合达摩院开源 RynnBrain 模型 + 乐云平台，线上学习更轻松

> ✅ **轻交付**：从概念理解到动手操作，无需购买硬件



## 怎么学



1. **了解具身智能**：概念与行业趋势（发展现状、当前挑战）
2. **具身智能的产品和应用形态**
   - 产品形态：机器人、智能运载装备、新型智能产品
   - 应用形态：人形机器人（双足/轮式）、四足机械狗、仿生机器人、机械臂等
3. **了解具身领域的研究方向和主要的技术路线**
   - VLA 模型
   - 世界模型
   - VLM 模型 + 世界模型：RynnBrain 模型（创新）👉🏻 体验 Demo
4. **乐云具身智能平台**
   - 常规方式：数据采集（真机+仿真）→ 模型微调与部署
   - 乐云平台功能详解：模型广场、数据管理、模型服务
   - 体验仿真调试：数据采集、模型调试、部署体验
5. **进阶探索**：学完课程，想“真机玩一玩”？
   - 参加乐云具身·SparkEdu训练营
   - 参考开源教程文档，自行购买教具复现
