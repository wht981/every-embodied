# 专题入口

本目录用于放置适合组队学习的完整实践专题。每个专题都应该包含清晰的学习目标、章节目录、可复现步骤和发布边界。

## 当前专题

- [OpenClaw 家庭物资助手](./02-OpenClaw家庭物资助手/README.md)
- [地瓜机器人新版教程桌宠](./03-地瓜机器人新版教程桌宠/README.md)

## 两个专题的关系

地瓜机器人新版教程桌宠侧重硬件入口、MagicBox 控制、语音、按钮、灯效和舵机交互。OpenClaw 家庭物资助手在这个基础上继续扩展到家庭物资管理、飞书通知、多维表格同步和前端展示。

如果读者第一次接触这套内容，建议先学习地瓜机器人新版教程桌宠，再学习 OpenClaw 家庭物资助手。

## 发布边界

这两个目录只保留可公开的教程内容、少量脚本和必要的忽略规则。公开时不要提交场景编辑器、缓存、模型权重、临时脚本、下载包、镜像文件、`.env` 和本机密钥。
